1. Put the all the files in a web server folder like htdocs/pdf/

2. Check the module availability of PDF::Reuse in the system.

3. If not available, install it

4. Run the url: http://localhost/pdf/sample_pdf.pl
 
   ( If you are putting the pdf in localhost )

5. A link will lead to view the pdf

6. A new pdf file will be created in the pdf folder

7. If you want to run in command prompt, run the file sample_pdf.pl

8. It will show HTML text. Check the newly created PDF file

9. KG has wrapper for PDF::Reuse

10. text_wrap is an internal module

11. BAS.pdf is a background pdf file. Content will be constructed over that.
